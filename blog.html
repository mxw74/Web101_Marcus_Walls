  2. Describe 2 pseudo-selectors and discuss what they are used for.

      3. Two pseudo-selectors I learned about in class today are hover and first-letter. The hover pseudo selector allows you to make text change whenever a person's mouse hovers over the text. The first-letter pseudo selector tells our css file to select every first letter. This is typically used to drop caps.
