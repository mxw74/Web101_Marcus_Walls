      3. The DOM is the document object model which is the idea that within each window is a document that contains our webpage and all the data that goes into it. This is important to understand because every webpage is designed using this model.

  3. Can you explain the difference between coding a website to be responsive and using a mobile-first strategy?

      4. When coding to make a websitee repsonsive this means that we use the @media function. This function allows us to define the min-width of our screens so that the .CSS design elements we apply only apply to screens with a min-width of which we defined. The mobile first strategy is used to make the base model of the website. This is typically easier because we are able to design our larger webpages much easier starting from the smallest pre-sets.
